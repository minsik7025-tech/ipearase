# 교실 대소동: 사라진 지우개 찾기

MECCHA CHAMELEON의 숨바꼭질/위장 페인팅 루프를 교실 테마로 옮긴 정적 웹 멀티플레이 게임입니다.

## 실행

```bash
npm test
npm start
```

## 배포

Vercel 정적 사이트로 배포할 수 있습니다.

```bash
npx vercel --prod
```

루트 `index.html`은 실제 게임 파일인 `사라진지우개찾기_배포용.html`로 쿼리스트링을 보존해 이동합니다. 따라서 `/?room=1234` 링크도 방 입장 링크로 동작합니다.

학생은 기본 주소로 접속한 뒤 열린 게임방 목록에서 방을 클릭해 입장할 수 있습니다. 누구나 방을 만들 수 있고, 방을 만들면 기존처럼 방 코드/QR/링크가 있는 로비 화면이 열립니다.

## Firebase

앱은 제공된 Firebase Realtime Database `earser` 프로젝트에 연결되어 있습니다. 클라이언트 웹 설정값은 공개 식별자이므로 HTML에 포함해도 되지만, 실제 접근 제어는 Realtime Database Rules에서 관리해야 합니다.

수업용 공개 테스트를 빠르게 시작하려면 Firebase 콘솔에서 테스트 모드를 사용할 수 있습니다. 공개 수업 링크를 장기간 운영한다면 최소한 `rooms` 경로만 허용하고 오래된 방을 정리하는 규칙/운영 절차를 두는 편이 안전합니다.
